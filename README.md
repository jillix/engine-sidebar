# engine-sidebar
jQuery.sidebar.js implementation for Engine.

## Configuration

 - `selector` (String): The sidebar element query selector.

## Documentation
The available functions are:

 - `open`
 - `close`
 - `toggle`

The data object should contain the `sel` field to select the sidebar element.
